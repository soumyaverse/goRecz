## Installation:

#### from source code
* `git clone https://github.com/burpOverflow/goRecz`
* `cd goRecz`
* `go build` or `go build -ldflags "-s -w"` 
Now you are ready to go...

#### using go get
* `go get github.com/burpOverflow/goRecz`
* binary file `goRecz` stored in `$GOPATH/bin`
